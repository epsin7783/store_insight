두 가지 분석 모드

파일 하나만 올리면 AI가 즉시 리포트를 생성합니다.

음성(회의) 분석

회의 녹음 파일을 업로드하면 전체 내용을 텍스트로 자동 변환하고, 핵심 문장 3줄을 추출하여 회의 내용을 빠르게 파악할 수 있습니다.

  • ✓ 음성 → 텍스트 자동 변환 (한국어 지원)
  • ✓ 핵심 문장 자동 추출 — 요약 3줄
  • ✓ MP3, WAV, M4A, OGG, FLAC 지원

영상(CCTV) 분석

CCTV 영상을 올리면 방문객 수·평균 체류 시간 등 경영 지표와 군집 밀집·쓰러짐 감지 등 안전 경고 타임라인을 차트로 시각화합니다.

  • ✓ 영상 자동 분석
  • ✓ 시간대별 방문객 통계 차트
  • ✓ 안전 경고 이벤트 타임라인
